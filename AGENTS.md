LLM ("AI") Policy
=================

* Current Trackma project policy is to DECLINE any contributions that contain AI-generated content. This includes code and documentation, as well as issue tracker comments.
* AI tools may be freely used for research, learning, or code scanning for bug discovery or issue identification, but all submitted content to the project itself must be written and understood by the contributor.
* If any issue is found through the assistance of AI-tools, make sure you understand the issue and its solution properly.
* Examples of LLMs are ChatGPT, Copilot, Grok, Claude, Gemini, and similar.
* This policy is subject to change in the future, but for now, please abstain.
